# Gamma Science Ontology

This directory defines the foundational ontology for Gamma Labyrinth scientific discovery. It provides a shared vocabulary for phenomena, cell types, mechanisms, and metrics used in the omission/predictive-coding modeling campaign.

## Structure

1. `core_entities.md` — High-level scientific targets and structural components.
2. `cell_types.md` — Biological classification and role definitions for neurons.
3. `mechanisms.md` — Hypothesized biophysical processes for omission and gain.
4. `metrics.md` — Standardized evidence labels and quantitative benchmarks.

## Usage

This ontology serves as the Control-plane reference for agents and researchers. When proposing missions, writing reports, or committing receipts, use the terms defined here to ensure scientific alignment and cross-repo consistency.

## Truth Status

`truth_mode: truth_safe_unverified`. Definitions here represent the scientific grounding and model rationale, not active biological truth or runtime state.
