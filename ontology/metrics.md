# Scientific Metrics & Benchmarks

## Standardized Metrics

- **Sparse Omission Spike Count**: Number of spikes during the omission window, used to quantify the strength of the omission response.
- **Post-Omission Gain Index**: A ratio or difference score comparing the response to the first post-omission stimulus against the pre-omission standard response.
- **LFP Spectral Shift**: Changes in power spectra (e.g., gamma vs. beta) during the omission and post-omission phases.
- **Biophysical Plausibility Score**: A qualitative or semi-quantitative measure of how well model parameters align with known biological ranges.

## Benchmarks

Missions must report these metrics to enable comparison across different model versions and scientific hypotheses. A result is `VERIFIED` only when these metrics meet the threshold defined in the specific mission task.
